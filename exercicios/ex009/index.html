Listas HTML

O HTML tem suporte para 3 tipos de listas. São elas:

  1. Listas Ordenadas
  2. Listas Não Ordenadas
  3. Listas de Definição

Listas Ordenadas

PAra criar listas ordenadas, sequenciadas, por exemplo 1, 2, 3..., usamos a tag OL (ordered list), e dentro dela usamos a tag LI para criar os itens.
Existe dentro da tag OL um parâmetro de escolha entre numeração, letra maiúscula, minúscula, algarismo romano maiúsculo e minúsculo: 1, A, a, XI, xi. Para usar este parâmetro adicionamos TYPE "" à tag OL, e dentro das aspas escolhemos a opção desejada.
DICA: Caso queira iniciar a lista com um número ou letra que não seja inicial padrão (1 ou A), adicionamos START "" após a tag TYPE"", e dentro das aspas inserimos o elemento que irá iniciar a lista.
Exemplo:

  1. Arroz
  2. Feijão
  3. Frango
OBS: O fechamento da tag OL é obrigatório (/OL). Já a tag LI não é obrigatório.

Listas Não Ordenadas

São listas em que não tem uma sequência obrigatória. Usa-se elementos como circulo pintado (disc), circulo contornado (circle) e quadrado (square).
O elemento disc é o padrão.
Para criar listas não ordenadas usamos a tag UL (unordered list), e para escolher o parâmetro adicionamos TYPE"" para selecionar entre disc, circel ou square.
Exemplo com square:

  • Arroz
  • Feijão
  • Frango

Juntando Listas

Para juntar uma lista com outra, por exemplo: uma lista não ordenada dentro de uma lista ordenada, usamos a tag dentro de outra tag (exemplo: UL dentro da tag OL). Esse recurso de juntar listas é muito utilizado em menus e índices.

DICA DE ATALHO: Para alterarmos várias tags ao mesmo tempo, clicamos e seguramos a tecla ALT, selecionamos todas as tags que serão alteradas, digitamos a tag correta e damos ENTER. Todas as tags serão alteradas juntas.

Listas de Definição

A lista de definição é menos utilizada do que as outras. Porém é muito importante para os mecanismos de busca, que priorizam as páginas com definição de listas para entregar ao usuário o melhor resultado/resposta. A lsta de definição é como um dicionário: usamos um termo e sua definição. Exemplo:

HTML (termo)
Linguagem de marcação para a criação do conteúdo de um site. (definição)

Para criarmos uma lista de definição usamos a tag DL (definition list), e para incluírmos o termo com definição usamos as tags DT (definition term) e DD (definition description) respectivamente.

Como no exemplo acima, os termos sempre ficam à esquerda e as descrições ficam deslocadas à direita.
