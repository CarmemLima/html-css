Citações e códigos

Fontes monoespaçadas são as mais recomendadas para a leitura de códigos. Isso porque alguns caracteres podem ficar um pouco ilegíveis se forem usados com fonte padrão. É o exemplo da letra l, t, i minusculas, essas letras ocupam menos espaço e junto de outras com mais espaço acabam não ficando muito legível. Sendo uma fonte monoespaçada não ocorre este problema, pois os caracteres tem o mesmo tamanho, o mesmo espaço de todos os outros, tornando assim o código mais legível.

Para usar a fonte monoespaçada, usamos a tag CODE, que indica que aquilo que foi digitado entre as tags se trata de um código fonte.

Indentar significa organizar o código com quebras de linhas, espaço e tabulações, ou seja, o texto indentado fica recuado dentro da página. Para indetarmos um código usamos a tag PRE envelopando a tag CODE que já havia sido digitada.
DICA: Para retirar as tabulações e espaços, ao invés de ir deletando um a um pode-se selecionar o código sem as tags e apertar SHIFT+TAB. E se quiser voltar os espaços e tabulaçoes é só clicar em TAB.

Citações Simples

Para usarmos frases com citações, geralmente usamos as aspas para indicar que é uma citação de outra pessoa. No entanto, ao colocar uma citação no HTML5, ao invés de só colcar as aspas é recomendado usar a tag Q de quote - que significa citar em inglês - para que tenha sentido e não apenas uma forma. Com esta tag Q a citação fica na mesma linha dando sequencia ao parágrafo.

Citações Completas

Para usar citações mais longas discriminando o código da origem de onde foi tirada a citação, usamos a tag BLOCKQUOTE. Com essa tag a citação fica recuada como um bloco separado do restante do parágrafo. Para fazer referência a esta citação podemos adicionar à mesma tag a palavra CITE "" (BLOCKQUOTE CITE""). Entre as aspas pode-se colocar o link de onde foi tirada a citação.
OBS: A referência do link não aparecerá na página, porém ficará como um dado armazenado para os mecanismos de busca (google, por exemplo).

Abreviações

Para criarmos abreviações usamos a tag ABBR. Por exemplo, na palavra HTML - que significa Hyper Text Markup Language - podemos envelopar com a tag ABBR TITLE "", e dentro das aspas colocar o significado da abreviação.
OBS: O significado da abreviação só irá aparecer quando passr o cursor na abreviação, que ficará com um sublinhado.

Texto Invertido

Segundo o professor Guanabara, a tag de texto invertido é a mais inútil dentro do HTML5. Mas, segundo ele também, um dia poderá ser que seja utilizado.

A tag para texto invertido é BDO (bi-directional overrride), e pode ser tanto BDO: RTL (para inverter o texto da direita para a esquerda), como pode ser BDO: LTR (para voltar o texto com leitura da esquerda para a direita). Por exemplo: A frase EU TE AMO fica EU TE AMO
