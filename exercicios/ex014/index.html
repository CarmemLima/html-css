Essa técnica separa o que é conteúdo do que é estilo, deixando o seu código HTML visualmente mais organizado, e de certa forma uma técnica mais vantajosa na organização e manutenção das alterações.
Para usarmos esse estilo interno usamos a tag STYLE dentro da taga HEAD, abaixo de TITLE. Dentro dessa tag STYLE criamos seletores para cada atributo que queremos utilizar como estilo. Esses seletores são chaves {} usados para cada elemento de conteúdo (ex: body, h1, h2, p, etc). E dentro das chaves usamos o atributo que queremos aplicar como estilo. Por exemplo, para alterar a cor de fundo inserimos STYLE>BODY{BACKGROUND-COLOR e escolhe a cor desejada}

Uma das vantagens nessa técnica é poder fazer alteração de todos os título h1 com uma única tag. No caso do estilo inline não era possível, pois era necessário fazer alteração em cada linha do h1.
