Texto marcado

Para usar um texto com estilo marcador de texto, podemos usar a tag MARK como na frase a seguir: texto marcado

No entanto, a tag MARK sempre deixará com a cor padrão amarela. Para alterar a cor do marcador de texto usamos uma tag do CSS dentro da tag MARK. Incluimos a tag STYLE, e entre as aspas incluimos a tag BACKGROUND COLOR e daí escolhemos a cor desejada. Veja o exemplo nesta frase: texto marcado

Se dentro da página houver vários textos ou frases com marcador de texto, recomenda-se usar uma tag generalizada para fazer a alteração automatica em todas. A tag de estilo (do CSS) fica abaixo de TITLE usando um seletor: adiciona a tag STYLE, dentro dela adiciona a tag MARK com o seletor {}. Por exemplo: texto marcado aqui e aqui e aqui. OBS: as palavras TEXTO MARCADO digitada no primeiro paragrafo deste tópico, também ficou com a cor alterada, pois ao alterar de forma generalizada o sistema irá alterar todas aquelas palavras ou frases que tiverem com a cor padrão amarela.

Texto grande e pequeno

usamos a tag BIG para usar palavras ou textos com fontes maiores, no entanto esta tag está obsoleta e por isso fica vermelha. Exemplo: texto grande

Para usar palavras ou texto com fontes menores usamos a tag SMALL. Exemplo: texto pequeno.

Texto deletado (tachado)

Para usarmos um texto com uma linha ao meio da frase indicando texto deletado, usamos a tag DEL. Exemplo: desconsidere esse texto

Texto inserido (sublinhado)

Para texto sublinhado usamos a tag U (não semântica) ou a tag INS (forma semântica). Exemplo 1: texto inserido com tag U. Exemplo 2: texto inserido com tag INS.

Texto sobrescrito

Para texto sobrescrito, ou seja, com caractere acima de outro caractere, usamos a tag SUP. OBS: a tag deve ser usada apenas entre o caractere a ser elevado. Exemplo: x2+3

Texto subscrito

Para texto subscrito, ou seja, com caractere abaixo de outro caractere, usamos a tag SUB. OBS: a tag deve ser usada apenas entre o caractere a ser elevado. Exemplo: H2O.
