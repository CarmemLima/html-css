Links e Âncoras em HTML5

Usando links externos

O fundamento básico do http e do html são os links.

Geralmente, e quase sempre, há um link dentro da página que está acessando. Os links são pontos clicáveis, que também são chamados de âncora ou hiperlink. Todo hiperlink é construído através de uma âncora dentro do código html.


CURIOSIDADE: O Google é o maior buscador de sites; e o segundo maior é o Youtube, que também é da Google.

Para criarmos links externos (links que levam para outra página) dentro de uma página, usamos a tag A HREF"". A de âncora, HREF de referência de hiperlink.

Dentro das aspas de HREF inserimos a URL da página que queremos abrir ao clicar.


IMPORTANTE: Para links externos (de outras páginas da web) é sempre recomendado que se use, após a tag A HREF"", a tag de parâmetro TARGET"" e REL"". Dentro de TARGET"" usamos o parâmetro _BLANK, que significa uma página em branco, ou seja, o link será aberto em uma nova aba e não na mesma página que está acessando. Dessa forma a sua página se manterá visível e aberta para o usuário retornar. Já na tag REL"", que quer dizer relativo à, usamos o parâmetro EXTERNAL, que significa externo, ou seja, que o link inserido é um link externo à sua página. Esses parâmetros são recomendados para que sua página esteja sempre aberta mesmo após visitar outros links externos.

Exemplo de link externo:
"Visite o meu site de moda fitness acessando Vitrine CL Fitness".

Links para download

Para criarmos um link para fazer download também usamos a tag A HREF"", mas, para inserirmos o link entre as aspas, o arquivo deve estar já salvo na pasta da página.

Também usamos um parâmetro dentro da tag AHREF"" que é o parâmetro DOWNLOAD. Dentro dele inserimos o arquivo a ser baixado. E também tem o parâmetro TYPE. Nesse parâmetro devemos utilzar as aplicações de tipos de midia, ou media types. Para isso devemos consultar no site iana.org/mediatypes qual a aplicação adequada para cada tipo de arquivo (pdf, zip, imagem, etc.)

Exemplo de link download foto e zip:

  • Clique aqui para baixar a foto
  • Clique aqui para baixar o arquivo zip


    • OBS: Em alguns navegadores o comando download do arquivo original não é atendido automaticamente (é o caso do Google, que não está baixando, só está abrindo na mesma guia). Nesse caso é necessario clicar com o botão direito sobre o link e clicar em baixar manualmente. Já no caso do arquivo zip ele já baixa automaticamente com um clique.
