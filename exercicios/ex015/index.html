Essa técnica separa em outra guia tudo o que for estilo deixando separado do código HTML usando apenas um link em cada página. Esse link irá atualizar automaticamente todas as páginas com a mesma configuração. Mas caso queira alterar apenas uma das páginas pode-se usar os outros estilos de css no mesmo arquivo. Nesse exemplo abaixo foi usado o estilo css externo e também o interno, no parágrafo do link para ira para a segunda página. Veja que ao ir para a segunda página, o link para voltar continua sem alteração.
